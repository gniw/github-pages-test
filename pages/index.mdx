# Welcome to GitHub Pages Test Documentation

This is a sample documentation site built with Nextra and Next.js Pages Router.

## Features

- **Static Site Generation**: Built with Next.js for optimal performance
- **Markdown Support**: Write documentation in MDX format
- **GitHub Pages**: Automatically deployed to GitHub Pages
- **PR Previews**: Preview documentation changes in pull requests

## Getting Started

To get started with this documentation:

1. Navigate through the sidebar
2. Read the documentation
3. Contribute by creating pull requests

## About Nextra

Nextra is a simple, powerful and flexible site generation framework built on top of Next.js.
